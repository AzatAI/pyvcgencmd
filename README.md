# Python Vcgencmd Wrapper

Python Wrapper for BCM (Raspberry Pi) vcgencmd

Still experimental.

## How to install 
`pip3 install pyvcgencmd`  
## How to use   
Quick example:  
``` python
from pyvcgencmd import Vcmd
```
